# SQL Databases

## ENG
A simple database design for a company selling CDs.
The existing AdventureWorks database was used to learn how to create databases.
We learnt how to create simple databases, how to add new records, how to join tables, and
how to retrieve specific data from a relational database.

## PL
Prosty projekt bazy danych dla firmy sprzedającej płyty CD.
Do nauki tworzenia baz danych wykorzystano już istniejącą bazę AdventureWorks.
Poznano, jak tworzyć proste bazy danych, jak dodawać nowe rekordy, jak łączyć tabele oraz w
jaki sposób pozyskiwać poszczególne dane z relacyjnej bazy danych.
